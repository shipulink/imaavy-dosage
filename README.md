# vyvgart-dosage
Imaavy IV infusion dosage calculator

This is a simple vibe-coded app (PWA) that takes a patient's mass in kg and calculates:
- Imaavy dosage in mg
- Imaavy dosage in mL
- Number and size of full vials to use
- mL from the last vial
- remaining mL and mg in the last vial (waste)
- mL to remove from a 250 mL IV normal saline bag in order to end up with 250 mL after adding the Imaavy dose.
